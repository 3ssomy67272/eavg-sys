# EgyptAir Virtual Group (EAVG) Systems Portal

Welcome to the official systems portal for the **EgyptAir Virtual Group (EAVG)**. This project is a Next.js application designed to manage the entire systems of EgyptAir Virtual Group operations, including VA website, recruitment system, crew center, discord bot, etc...

## Requirements

Before you begin, ensure you have the following installed:
- **Node.js**: Version 18.17.0 or higher.
- **Package Manager**: npm (comes with Node), yarn, or pnpm.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository (or your fork).
  ```bash
   git clone https://github.com/3ssomy67272/eavg-sys
   ```
2 Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server and see the app in action:

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the result.
